# 🎵 Калимба — Веб-симулятор музыкального инструмента

> Интерактивный симулятор 17-клавишной калимбы, работающий прямо в браузере. Звук генерируется в реальном времени с помощью Web Audio API, без использования внешних семплов или зависимостей.

## ✨ Функционал

- 🎹 **17 настроенных пластин** в тональности До мажор (C Major)
- 🎛️ **Реалистичный синтез звука**: используются несколько гармоник, FM-модуляция и точные огибающие (ADSR), имитирующие характерное тёплое и металлическое звучание настоящей калимбы
- ⌨️ **Управление клавиатурой**: полная поддержка английской (EN) и русской (RU) раскладок
- 🖱️ **Клик и тач**: играйте мышью на ПК или пальцем на мобильных устройствах
- 🌊 **Реверберация**: переключатель объёмного пространственного звучания
- 🔊 **Регулятор громкости**
- 🎶 **Демо-мелодия**: встроенный автовоспроизводимый этюд для быстрого ознакомления с инструментом
- 🌌 **Анимированный фон**: плавающие градиенты и мерцающие звёзды на чистом CSS
- 📱 **Адаптивный дизайн**: интерфейс корректно отображается на экранах любого размера

## 🚀 Как запустить на своём компьютере

Приложение не требует сборки, установки Node.js или запуска серверов. Всё работает из одного файла:

1. **Скачайте файл** `index.html` из репозитория (или клонируйте весь репозиторий).
2. **Откройте файл** в любом современном браузере (Chrome, Firefox, Safari, Edge) двойным кликом или перетаскиванием в окно браузера.
3. **Разрешите звук**: современные браузеры блокируют автовоспроизведение аудио до первого взаимодействия со страницей. Просто нажмите любую клавишу или кликните по пластине.
4. **Играйте!** Используйте клавиши клавиатуры (EN: `Z-M`, `Q`, `A-L` / RU: `Я-Ь`, `Й`, `Ф-Д`) или кликайте по пластинам мышью/тачем.

> 💡 *Совет:* Для лучшего звучания используйте наушники или внешние колонки. Браузерный `Web Audio API` раскрывает весь потенциал синтеза именно на качественных аудиовыходах.

## 🛠 Технологии

- `HTML5` + `CSS3` (Flexbox, CSS-анимации, переменные, адаптивность)
- `Vanilla JavaScript (ES6+)`
- `Web Audio API` (синтез звука, фильтры, свёрточная реверберация, динамическое управление огибающими)
- Без сторонних библиотек и зависимостей
